ORIGIN

Damaged Battery Inc. is a software company conceptualized with the intent of long term value creation for customers within the cryptocurrency space. It provides optimal solutions by enabling investors, traders, and businesses to secure quick data generation coupled with ease of application.

The main product is the CryptXcel app for the Mac OS platform. It is a utility tool that incorporates data import and price tracking functionalities into Microsoft Excel spreadsheets, for effective monitoring and best performance in portfolio management.
