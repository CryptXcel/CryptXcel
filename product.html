Unleash the Power of CryptXcel

CryptXcel's primary focus lies in extracting current and up-to-date cryptocurrency price data from Coingecko and CoinMarketCap.

This data can be expressed in multiple supported fiat currencies, cryptocurrencies, or bitcoin units and is seamlessly imported into Excel for Mac spreadsheets via API key requests.

Making API calls and parsing data can be an overwhelming process. CryptXcel resolves that by allowing users to conveniently retrieve and integrate that price data with a click of a button.

All Coingecko and CoinMarketCap API requests are fully automated by CryptXcel.

Registration for an API key with Coingecko is not required. CoinMarketCap API requests require a user-specific API key that can be obtained by registering for a basic plan.

4 MODES AND FEATURES

Coingecko Full Mode

  • Choose to display 100 to 1000 coins with one request.

    This Free API has a rate limit of 50 calls/minute.

    Coin prices can be priced and displayed in 4 fiat currencies and 2 cryptocurrencies simultaneously.

Coingecko Basic Mode

  • Choose to display selected coins with Coingecko's ID system.

    Basic mode will display selected coins in 1 fiat currency and 1 cryptocurrency or concurrently.

CoinMarketCap Full Mode

  • An API call rate limit of 30 requests a minute.

    1 credit count or 1 API call request for every 200 coins.

    Up to 5000 coins may be called in one request.

    Coin prices can be priced and displayed in 4 fiat currencies and 2 cryptocurrencies simultaneously.

CoinMarketCap Basic Mode

  • Users may choose to display 5 selected coins with coin symbols.

    Basic mode will display selected coins in 1 fiat currency and 1 cryptocurrency or concurrently.
